# 🎵 Spotify Clone (Mimicking the spotufy web player)
This is a **Spotify-inspired web app**  built using **HTML, CSS, and JavaScript** as part of my learning/project work.

I tried to recreate the look and feel of Spotify while also adding some basic music player functionality.

---

## 🚀 What this project can do

* Play songs ▶️
* Pause / Resume ⏸
* Next / Previous tracks ⏭
* Show trending songs, artists, albums, and charts
* Clean Spotify-style UI 🎨
* Separate login page with social buttons 🔐

---

## 🛠️ Web Tech I used

* HTML
* CSS
* JavaScript (no frameworks, just pure JS)
